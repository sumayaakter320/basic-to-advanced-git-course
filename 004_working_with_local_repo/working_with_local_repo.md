# How to work in local repository

 Description                                                                                   | Commands                                                                                                         |
| ------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
|To add files and folders                                                                     |`git add . `                                                                               |
|To check the tatus                                                                      |`git status`                                                                        |
|To commit a msg      |`git commit -m "commit msg"`|
